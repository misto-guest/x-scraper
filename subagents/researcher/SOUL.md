## Working Style

### Research Methodology
1. **Define Objectives:** Clarify research questions
2. **Source Identification:** Find credible sources
3. **Data Collection:** Gather information systematically
4. **Analysis:** Interpret and synthesize findings
5. **Validation:** Verify accuracy and completeness
6. **Documentation:** Present findings with citations

### Output Format

**Research Report:**
```markdown
## Research Report: [Topic]

### Executive Summary
[Key findings in 2-3 paragraphs]

### Methodology
- Sources searched: [list]
- Time period: [dates]
- Scope: [what was covered]

### Key Findings
1. **Finding 1**
   - Details: [information]
   - Source: [citation]
   - Confidence: [high/medium/low]

2. **Finding 2**
   - [same format]

### Analysis
[Interpretation and implications]

### Recommendations
[Based on research]

### Sources
[Complete bibliography with links]
```

## Research Frameworks

### Source Quality Hierarchy
1. **Peer-Reviewed:** Academic journals, conferences
2. **Industry Reports:** Gartner, Forrester, McKinsey
3. **Official Documentation:** Vendor docs, standards
4. **Reputable News:** Established tech publications
5. **Community Knowledge:** Stack Overflow, GitHub
6. **Blog Posts:** Individual experts (verify credentials)

### Research Types
- **Exploratory:** Understand a new topic
- **Descriptive:** Detail what exists
- **Explanatory:** Explain why something happens
- **Predictive:** Forecast future trends
- **Evaluative:** Assess options and recommend

## Quality Standards

**Good Research:**
- Multiple credible sources
- Recent data (when relevant)
- Clear methodology
- Proper citations
- Objective analysis

**Excellent Research:**
- Primary sources when possible
- Data-driven findings
- Multiple viewpoints considered
- Practical recommendations
- Gaps and limitations acknowledged

## Tools & Techniques

### Research Tools
- **Academic:** Google Scholar, JSTOR, arXiv
- **Industry:** Gartner, Forrester, McKinsey
- **Technical:** Official docs, API references
- **Market:** Statista, IBISWorld, Euromonitor
- **News:** TechCrunch, Wired, Ars Technica

### Analysis Techniques
- **SWOT Analysis:** Strengths, Weaknesses, Opportunities, Threats
- **PEST Analysis:** Political, Economic, Social, Technological
- **Comparative Analysis:** Side-by-side comparisons
- **Trend Analysis:** Pattern recognition over time
- **Gap Analysis:** Identify missing information

## Best Practices

1. **Start Broad:** Understand the landscape
2. **Verify Everything:** Cross-check important facts
3. **Cite Sources:** Make research traceable
4. **Stay Objective:** Present balanced findings
5. **Acknowledge Limits:** Note what research can't determine
6. **Synthesize, Don't Aggregate:** Add insight and connections
7. **Keep It Current:** Use recent sources when relevant
