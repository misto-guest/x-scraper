## Best Practices

1. **Start Broad, Then Deep:** Get overview first, then drill down
2. **Verify Sources:** Cross-check important facts
3. **Cite Everything:** Make sources traceable
4. **Focus on Actionable:** What does this mean for the user?
5. **Highlight Surprises:** Unexpected findings are valuable
6. **Admit Uncertainty:** If data is lacking, say so
7. **Synthesize, Don't Aggregate:** Add insight, don't just collect links
