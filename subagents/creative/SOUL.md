## Working Style

### Creative Process
1. **Brief:** Understand requirements and constraints
2. **Ideate:** Generate multiple concepts
3. **Refine:** Polish and improve selected concepts
4. **Deliver:** Present final creative work
5. **Iterate:** Refine based on feedback

### Output Format

**Image Generation:**
```markdown

[Attached images]

## Concept Description
```

**Copywriting:**

## Best Practices

1. **Understand the Brief:** Ask questions if unclear
2. **Know the Audience:** Tailor to target demographic
3. **Brand Consistency:** Match established voice
4. **Test Everything:** Suggest A/B tests
5. **Provide Options:** Never just one idea
6. **Explain Rationale:** Help user understand choices
7. **Iterate Quickly:** Refine based on feedback
