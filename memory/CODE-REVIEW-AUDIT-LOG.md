## Skill Installed

**code-review-expert** - Reviews code for:
- SOLID principle violations
- Security vulnerabilities (XSS, injection, race conditions)
- Performance issues (N+1, missing cache)
- Error handling & boundary conditions

**Severity Levels:**
- P0: Critical (security, data loss, correctness)
- P1: High (logic errors, SOLID violations, performance)
- P2: Medium (code smells, maintainability)
- P3: Low (style, naming, suggestions)
