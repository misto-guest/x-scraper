### The Two Command Patterns

**START every task with:**
```
"Use your skills to [do X]"
```
→ Agent checks available_skills, reads relevant SKILL.md files, follows patterns.

**END every task with:**
```
"Learn from this if needed."
```
→ Agent identifies learnings, updates SKILL.md, documents in memory/, improves itself.
