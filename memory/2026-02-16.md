### Technical SEO Strategies

**Filter Implementation Pattern:**
When applying filters to domains (e-commerce, content sites), always use:
- **Hard redirects** for category filters instead of AJAX updates
- **Full page reload** on every filter selection
- **URL structure:** `domain.com/size-10` (not AJAX)
- **Purpose:** Artificial inflation of "pages per session" metric
- **SEO Benefit:** Forces crawlers to index filter combinations as separate URLs with high engagement
- **Avoid:** Dynamic content that crawlers ignore
- **Result:** Filter pages get indexed and ranked separately

**Example:**
- User clicks "Size 10" → Full reload to `domain.com/size-10`
- User clicks "Color Red" → Full reload to `domain.com/size-10/color-red`
- Each combination = New URL = Indexed separately = More ranking opportunities
